1. Чому typeof null повертає "object"?
   Один з початкових багів з самого створення мови. Тег object 0, а тег null також 0.
2. Яка різниця між let , const і var ?
   let - змінна, яку можна змінювати
   const - константа, значення змінити неможливо
   var - застарілий, вище ризик щось зламати
3. Що станеться, якщо викликати неоголошену змінну в strict mode?
   Виникне помилка ReferenceError.
4. Як працює перетворення типів у JavaScript?
   Ти самостійно їх викликаєш або JS самостійно це робить коли працює з різними видами данних.
5. Які переваги strict mode?
   Забороняє випадкове створення глобальних змінних, що запобігає забрудненню пам'яті та важкодоступним багам.
